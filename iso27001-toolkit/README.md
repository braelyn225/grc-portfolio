# ISO 27001 Toolkit

This toolkit provides working documents to support ISO/IEC 27001 readiness and implementation.

## Contents
- **Controls Mapping:** Map Annex A controls to implemented controls and evidence.
- **Statement of Applicability (SoA):** Document included/excluded controls and justification.
- **ISMS Scope:** Define the boundaries of the ISMS.
- **Risk Assessment Methodology:** Document the approach to risk assessment.
