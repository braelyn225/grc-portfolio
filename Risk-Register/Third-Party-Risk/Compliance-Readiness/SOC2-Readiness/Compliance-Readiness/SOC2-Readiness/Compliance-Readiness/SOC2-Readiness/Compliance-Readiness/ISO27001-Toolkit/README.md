# ISO/IEC 27001 Toolkit

## Purpose
ISO/IEC 27001 is an international standard for establishing, implementing, maintaining, and continually improving an Information Security Management System (ISMS).

This folder demonstrates core ISO 27001 readiness artifacts commonly developed by GRC teams.

## Key ISO 27001 Components
- Risk assessment and risk treatment
- Selection of applicable security controls
- Policy and documentation management
- Continuous improvement and governance

## Artifacts Included
- **Statement-of-Applicability-Sample.md:** Control selection and justification
- **Risk-Treatment-Plan-Sample.md:** Risk remediation tracking

These artifacts align with ISO/IEC 27001:2022 requirements and Annex A controls.
