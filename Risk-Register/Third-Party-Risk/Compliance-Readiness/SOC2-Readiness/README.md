# SOC 2 Evidence Request List (Sample)

## Governance
- Approved information security policies
- Organizational roles and responsibilities documentation

## Access Controls
- User access listings
- MFA enforcement screenshots
- Access review records

## Change Management
- Change tickets and approvals
- Code review evidence
- Deployment logs

## Incident Response
- Incident response plan
- Incident tickets (sanitized)
- Post-incident reviews

## Vendor Management
- Vendor inventory
- SOC reports or questionnaires
- Contract security clauses
